The Big Three

These three methods handle 90% of array transformations:

map()

Transform every element in an array:

const names = users.map(user => user.name);
// ["Alice", "Bob", "Charlie"]

filter()

Keep only elements that pass a test:

const adults = users.filter(user => user.age >= 18);

reduce()

Collapse an array into a single value:

const total = prices.reduce((sum, price) => sum + price, 0);

Handy Helpers

  • find() — returns the first match
  • some() — true if any element passes
  • every() — true if all elements pass
  • includes() — checks if a value exists
  • flat() — flattens nested arrays

Chaining

The real power comes from chaining methods together:

const result = users
  .filter(u => u.active)
  .map(u => u.name)
  .sort();
When in doubt, reach for map and filter before writing a for loop.
