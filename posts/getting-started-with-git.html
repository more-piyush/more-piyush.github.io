Why Git?

Git is the most widely used version control system in the world. Whether you're working solo or on a team, Git helps you track changes, revert mistakes, and collaborate effectively.

Setting Up

First, install Git from the official website. Then configure your identity:

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Basic Workflow

The core Git workflow is simple:

  1. Initialize a repository with git init
  2. Stage changes with git add
  3. Commit changes with git commit -m "message"
  4. Push to a remote with git push

Branching

Branches let you work on features in isolation. Create a branch with git checkout -b feature-name, do your work, then merge it back into main when ready.

Pro tip: commit often and write meaningful commit messages. Your future self will thank you.

Next Steps

Once you're comfortable with the basics, explore rebasing, cherry-picking, and Git hooks to level up your workflow.
